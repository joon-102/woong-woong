## 시작하기
### 필수 설치  
- [Node.js](https://nodejs.org/)

#### 저장소 복제
```sh
git clone https://github.com/joon-102/woong-woong
cd woong-woong
```
#### 의존성 설치
```sh
npm install
```
#### 실행  
```bash
npm start
```
이제 http://localhost:3000 에서 앱이 시작됩니다!
